# Manifesto: Desempenho Não é Performance (dnp)

Este manifesto estabelece a base teórica e ética para tudo o que construímos no monorepo **dnp**. É um chamado para repensar nossa relação com a tecnologia, o trabalho e o crescimento.

## O Paradoxo Materialista

Vivemos em um sistema que exige "performance" constante: mais cliques, mais usuários, mais lucro, mais código. No entanto, essa performance é muitas vezes vazia, uma casca estética que esconde a fragilidade material e a exploração.

O **Paradoxo Materialista** do dnp é: usar as ferramentas do capitalismo (como a plataforma Roblox e IAs proprietárias) para construir as sementes de sua superação. Reconhecemos as contradições e as usamos como motor de transformação através do **Materialismo Dialético**.

## Desempenho vs. Performance

Para nós, os termos não são sinônimos. O anglicismo "performance" foi corrompido pelo capital para significar "eficiência exploratória".
- **Performance:** É a métrica do sistema. É o crescimento infinito, o código inchado (bloat), e a busca por KPIs vazios.
- **Desempenho:** É a realização material. É o cumprimento de um propósito social e ecológico. É o código resiliente, a soberania tecnológica e o equilíbrio sistêmico.

## Pilares do Manifesto

### 1. Decrescimento (Degrowth) como Pilar
Não buscamos ser a "maior" empresa de jogos. Buscamos ser a mais resiliente. O decrescimento significa priorizar a utilidade social e a saúde planetária sobre o lucro. Em nossos jogos, isso se traduz em mecânicas que recompensam o equilíbrio, não o acúmulo.

### 2. Soberania Tecnológica
Buscamos deter os meios de produção digital. Isso significa usar ferramentas como Rojo e Git para manter o controle sobre nossa lógica, e documentar cada passo para que o conhecimento não seja privatizado.

### 3. Fricção Consciente (Conscious Friction)
Rejeitamos a "fluidez" que leva ao consumo irracional. Nossas interfaces e mecânicas devem convidar à reflexão. O usuário deve entender as engrenagens do sistema, não apenas consumir o produto final.

### 4. Horizonte Eco-socialista
O dnp é um projeto político. Acreditamos que a tecnologia deve servir para reerguer o mundo dos escombros do capitalismo. Nossas ferramentas e jogos são exercícios de imaginação radical para um futuro comunista e ecológico.

## Filosofia dnp

O manifesto é a alma da nossa **Filosofia dnp**. Ele nos lembra que cada linha de código é um ato político. Desenvolver no Windows, usar o PowerShell em modo admin, comandar uma IA — tudo isso são interações com a matéria que devem ser guiadas por uma ética de libertação, não de submissão.
