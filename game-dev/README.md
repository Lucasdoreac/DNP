## Português

Bem-vindo ao subprojeto de desenvolvimento de jogos do ecossistema **dnp (Desempenho Não é Performance)**. Este diretório é dedicado à criação soberana de experiências no Roblox, utilizando ferramentas profissionais e uma filosofia anticapitalista.

### 🎯 Objetivos
- **Soberania do Código:** Uso de Rojo e TypeScript para manter a lógica em ambiente local e versionada via Git.
- **Filosofia dnp:** Implementação de mecânicas de jogo baseadas em decrescimento, equilíbrio e fricção consciente.
- **Aprendizado Contínuo:** Trilha de aprendizado completa estruturada para ingestão em IAs como NotebookLM.

### 📚 Trilha de Aprendizado
A trilha completa está disponível em [docs/learning-path/](./docs/learning-path/index.md), contendo:
- Fundamentos de Roblox e Luau.
- Configuração de Rojo e Roblox-TS.
- Manifesto dnp e Design Anticapitalista.
- Materiais multimídia gerados (Podcasts, Vídeos, Flashcards).

## English

Welcome to the game development subproject of the **dnp (Performance is Not Performance)** ecosystem. This directory is dedicated to the sovereign creation of Roblox experiences, using professional tools and an anti-capitalist philosophy.

### 🎯 Goals
- **Code Sovereignty:** Use of Rojo and TypeScript to keep logic in a local environment and versioned via Git.
- **dnp Philosophy:** Implementation of game mechanics based on degrowth, balance, and conscious friction.
- **Continuous Learning:** A complete learning path structured for ingestion into AIs like NotebookLM.

### 📚 Learning Path
The full path is available at [docs/learning-path/](./docs/learning-path/index.md), containing:
- Roblox and Luau Fundamentals.
- Rojo and Roblox-TS Configuration.
- dnp Manifesto and Anti-capitalist Design.
- Generated multimedia materials (Podcasts, Videos, Flashcards).
