# Track Specification: Learning Path (Portuguese)

## Goal
Create a comprehensive, structured learning path in Brazilian Portuguese covering Roblox development, Rojo/TypeScript integration, and the "dnp" (Desempenho Não é Performance) philosophy. This content is intended to be processed by NotebookLM to generate multimedia learning materials (videos, podcasts, slides).

## Requirements
- All content must be in Brazilian Portuguese.
- Content must cover:
  - Roblox Studio and Luau fundamentals.
  - Rojo and TypeScript (Roblox-TS) setup.
  - "dnp" philosophy: Degrowth, Anti-capitalism, Sovereignty.
  - Server analysis and ethical monetization.
- Format: Markdown files structured for NotebookLM ingestion.
- Language: Dialectical Materialism (as per Product Guidelines).

## Target Artifacts
- A series of markdown documents in `docs/learning-path/`.
- A master index/summary for NotebookLM.
